## Installation

ZumoKit is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'ZumoKit', :git => 'https://github.com/zumo/zumokit-ios.git', :branch => 'release/1.0.3'
```
